Documentation - Algorithme Box Blur

Analyse de l'effet d'atténuation par calcul de moyenne locale

1. Pourquoi utiliser le Box Blur ?

  • Ultra-rapide (O(1)) : Contrairement au flou gaussien (lourd en multiplications décimales), le Box Blur n'utilise que des additions et une division (calcul de moyenne). Combiné avec une image intégrale (Summed-Area Table), le temps de calcul reste identique, que le rayon du flou soit de 3 ou de 300 pixels.
  • Alternative au Flou Gaussien (Astuce des 3 passes) : Un seul Box Blur peut paraître un peu grossier/"carré". Mais si vous l'appliquez 3 fois de suite, le résultat visuel est quasi-identique à un vrai flou gaussien, tout en étant 3 à 5 fois plus rapide à calculer.
  • Nettoyage du bruit : Il sert de filtre de pré-traitement. Appliquer un Box Blur léger élimine les pixels parasites (le bruit) avant de lancer des algos plus complexes comme les détecteurs de contours (Sobel/Canny), évitant ainsi les fausses détections.

2. Exemples d'implémentation (PureBasic)

Option A : Utilisation Directe (Fonction Standard)

Idéal pour les appels simples et rapides avec passage de paramètres en arguments.

IncludeFile "filtres.pbi"
UseModule filtres

If OpenWindow(0, 0, 0, 800, 600, "Exemple...", #PB_Window_SystemMenu | #PB_Window_ScreenCentered)
  
  SourceImage = 1
  CibleImage  = 2
  AucunMasque = 0
  RayonX      = 5
  RayonY      = 5
  Passes      = 1 ; <--- Passez à 3 pour simuler le flou gaussien
  ModeBord    = 0 ; Traitement standard des bords
 
  file$ = OpenFileRequester("Image","","",0)
  If load_image_32(SourceImage , file$) = 0
    End
  Else
    ResizeImage(SourceImage, 800, 600)
  EndIf
  
  CopyImage(SourceImage,CibleImage)

  BoxBlur(SourceImage, CibleImage, AucunMasque, RayonX, RayonY, Passes, ModeBord)
  
  StartDrawing(WindowOutput(0))
  DrawImage(ImageID(CibleImage) ,  0 , 0)
  StopDrawing()
  
  Repeat
    Event = WaitWindowEvent()
    Select Event 
      Case #PB_Event_Gadget
        Select EventGadget()
          Case 1 
            CloseWindow(0)
            End  
        EndSelect
    EndSelect

  Until Event = #PB_Event_CloseWindow
EndIf

3. Le Filtre : Comment ça marche ?

Le filtre de Box Blur (ou flou par boîte) est un traitement graphique qui adoucit les images. Son rôle fondamental est de lisser les détails, d'estomper les cassures nettes et d'effacer les imperfections visuelles (le bruit numérique) pour créer un effet de flou uniforme.

Pour obtenir ce résultat de la manière la plus simple et efficace possible, le filtre utilise un principe de calcul à la portée de n'importe quel ordinateur : la moyenne mathématique.

Le principe du mélange des couleurs

Imaginez que chaque pixel de votre photo soit une petite goutte de peinture fraîche. Pour flouter un point précis de l'image, le filtre va prendre ce point, regarder la couleur des points qui l'entourent, et mélanger le tout pour obtenir une teinte intermédiaire parfaitement homogène.

La boîte de voisinage (Le Masque)

Pour faire son mélange, l'ordinateur dessine une boîte virtuelle (souvent un carré de 3x3 ou 5x5 pixels) centrée sur le point qu'il souhaite modifier.

La formule logique visuelle :

Nouvelle_Couleur = Somme_des_Couleurs_du_Carré / Nombre_Total_de_Pixels

Dans cet exemple, le pixel central (indigo) fusionne avec ses 8 voisins (bleus). On additionne l'intensité de ces 9 cases, puis on divise le tout par 9. Aucun pixel n'a plus d'importance qu'un autre.

Logique de traitement globale (Pseudo-code)

Voici le cheminement logique simplifié que suit l'algorithme pour balayer et flouter toute la surface d'une image :

Pour chaque ligne de la photo
    Pour chaque pixel de cette ligne
        
        Somme_Rouge = 0
        Somme_Vert  = 0
        Somme_Bleu  = 0
        Compteur    = 0
        
        // Ouvrir la boîte tout autour du pixel courant
        Pour chaque pixel situé à l'intérieur du carré de voisinage
            Somme_Rouge + Pixel_Voisin_Rouge
            Somme_Vert  + Pixel_Voisin_Vert
            Somme_Bleu  + Pixel_Voisin_Bleu
            Compteur    + 1
        Next
        
        // Calculer la couleur moyenne finale
        Moyenne_R = Somme_Rouge / Compteur
        Moyenne_V = Somme_Vert / Compteur
        Moyenne_B = Somme_Bleu / Compteur
        
        // Ecrire le résultat sur l'image finale floutée
        Dessiner_Pixel_Flou(x, y, Moyenne_R, Moyenne_V, Moyenne_B)
        
    Next
